# 账单管理前端

React + Ant Design 前端应用

## 启动步骤

### 1. 安装依赖

```bash
cd frontend
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

前端将在 `http://localhost:3000` 启动

### 3. 确保后端运行

确保后端 FastAPI 服务在 `http://localhost:8000` 运行

## 页面说明

- `/dashboard` - 账单小助手（仪表盘）
- `/bills` - 账单管理
- `/analysis` - 消费分析
- `/ai-assistant` - AI助手
- `/recommendations` - 金融产品推荐
- `/community` - 社区

## 主题

使用 WeBank 微众银行红白蓝主题：
- 主色调：`#E02020` (WeBank 红)
- 辅助色：`#1890ff` (蓝色)
- 背景：`#f5f7fa`
